# Lian Li USB protocol
## Packets structure
### Color mode switch packet:
| Start (?) | Port ID | First mode byte | Second mode byte |
|----------|:-------------:|:-------------:|:-------------:|
| E0 | 10-14 | 00-? | 00-? |

### Command packet:
| Start (?) | Channel | Command | Argument |
|----------|:-------------:|:-------------:|:-------------:|
| E0 | 10-14 | 10-24 | 00-31 | 00-FF |

## Known bytes
`BaseByte` (start byte) = 0xE0 <br />
`DefaultChannelByte` = 0x10 <br />
